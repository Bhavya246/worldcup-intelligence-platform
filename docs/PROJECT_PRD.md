# WorldCup Intelligence Platform

## Vision

Create a professional football analytics platform capable of predicting FIFA World Cup 2026 match outcomes.

---

## Objective

Predict match outcomes using Elo ratings and machine learning models while providing explainable probabilities through a live dashboard.

---

## Data Sources

### Historical Data
- StatsBomb Open Data
- OpenFootball

### Team Ratings
- ClubElo

### Live Data
- Zafronix World Cup API
